`Extend` is to import a module's methods to a class.
`Include` is to import a module's methods to all instances of a class.
`require` imports dependencies to the current file

`Super` copies the superclass instance variables to the current class.

`self` when used in the code for a class, for example when used in a method, refers to the instance itself.

`attr-reader` allows you to call an instance variable as a method on an instance of a class.
`attr-writer` allows you to set this instance variable
`attr_accessor` does both.
